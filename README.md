# STUDENT HABITS PERFORMANCE PREDICTION
**Likely positive predictors of performance**
Variables that typically show positive correlations with grades:
-Higher study hours
-Better sleep quality/duration
-Higher class participation
-Good attendance
-Strong health score
**Likely negative predictors**
Variables commonly found to correlate negatively with student performance:
-Excessive social media usage
-High stress levels
-Very high extracurricular load (if not balanced)
# Key Insights of it..
-Student habits have a measurable and strong impact on academic performance.
-Study hours, sleep, attendance, and participation are the strongest positive predictors.
 -Stress level and social media use have negative impacts on grades.
 -A tree-based ensemble model (likely XGBoost or Random Forest) is the best predictor.
-The best model was saved as best_model.pkl for future deployment.
